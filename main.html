    Projects

    Muse

    Muse is a Discord bot that sends music based on user-inputted countries and decades and keeps track of top search terms and recently liked songs in a server. This bot uses Discord's API to send messages based on an user's input that it reads. Muse uses Musicbrainz to search up songs from a certain country and decade and searches YouTube for a link for that song. Additionally, it pulls up a list of countries that existed during decades after the 1900s and top searched and recently liked songs among all servers Muse has been invited to.
    All commands have a '+', followed by the valid terms 'help', 'list', 'song', and 'top'. More information about the bot and its server invite can be found in the links below.

    MazeBot

    MazeBot is maze traversal robot that uses LIDAR sensors to navigate through any given maze. It is a collaborative project of students of TURTLE robotics from computer science, electrical engineering, mechanical engineering, and related majors. Mazebot is run using a Nvidia Jetson computer and the final product is ensured to have perfect accuracy by using data gathered by previous trials. This project is currently a work in progress.

    My Luggage Handler

    My Luggage Handler (MLH) is a luggage handling system comprising of RFID chips, an SMS messaging system, and a web app that displays the status of a traveller's bags. It was made using backend and frontend programming languages and an Arduino, and the prototype simulates the system for only one luggage. I made MLH with 3 of my friends for Major Hacking League's Late Night Hacks Hackathon, and we won the most creative use to Twilio for this project.
    More information about the My Luggage Handler system and my contribution to it can be found in the links below.

    Tabs

    Tabs is a web app that functions as a bulletin board for airports where passengers can post their lost items for others to find around the terminal and allows them to update the status of their items. This app was built using HTML and vanilla CSS and Javascript, and it was submitted to the AA Challenge during Tamuhack 2023. I coded the UI of Tabs and created a mock database storing user data, using JSON, which is then used for confirming login and displaying users' feeds.

    ARMv8 Processor

    This project is a replica of the ARMv8 processor in Verilog for ECEN 350, a course on computer architecture. It uses various logic and controls units that manage 64 bits of local data based on inputted assembly code.
